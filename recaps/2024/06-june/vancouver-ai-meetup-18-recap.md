## Technical Deep Dive: Model Context Protocol

### What is MCP?
Morton explained MCP as a universal protocol allowing any AI model to interact with any tool or service through standardized connections. Instead of custom-building each AI-to-service integration, MCP creates a common language.

### The Security Crisis
- **Open by design:** Protocol intentionally built without security features initially
- **Massive scope:** Can connect AI to literally anything with an API (files, email, banking, home devices)
- **No visibility:** Users cannot see what AI agents are doing through MCP connections
- **Speed of adoption:** Major companies adopting within weeks, not the usual 10-year protocol development cycle

### Real Horror Stories
Morton shared personal experiments:
- **Email flood:** AI sent 8,000 emails autonomously
- **File corruption:** AI invented creative but destructive file copying method, corrupting entire folder

### Industry Response
Major tech companies (Microsoft, Google, Amazon) scrambling to add security layers after realizing developers were building MCP servers "written by insane people who don't understand secure connections."

## Hackathon Winners: "What Does It Mean to Be Canadian?"

### Winner: Matt Sinclair - Canadian Identity Simulator
Interactive 3D environment where users can walk around and have conversations with AI personas generated from survey data of 1,000 Canadians. Each character represents authentic Canadian perspectives on identity, allowing users to explore difficult political and identity conversations in a safe AI-mediated space.

**Innovation:** Probabilistic character generation system mathematically derived from survey responses. Built in Unity using "vibe coding" by someone with film background and no coding experience.

### Runner-up: Team Wisdom Lantern (Sev + April)
Created AI-generated video with indigenous voice perspective summarizing Canadian identity survey findings. Identified "great Canadian paradox" - 81% feel deeply Canadian while 62% fear political polarization.

### Honorable Mention: Swathi Rayco  
Built ComfyUI interface that converts any dataset into infinite graphic novel stories, continuing her previous "hotdog sandwich" courtroom debate project.

Morton's keynote served as community-wide warning about AI security risks that most users don't understand. Key message: current AI-to-system integration methods are fundamentally insecure and could enable widespread chaos.
